## Install

Ensure `Allow Remote Automation` is ticked under the `Develop` menu in Safari. You can eneable the Safari `Develop` menu in Safari preferences.

```
pip3 install selenium
```

## Usage

```
python3 click.py URL ELEMENT_ID NUM_CLICKS
```
